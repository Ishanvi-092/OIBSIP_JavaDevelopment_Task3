# Task 3 – ATM Interface

## Objective

The objective of this project is to design and implement a **console-based ATM interface using Java** that simulates basic banking operations. The system requires users to log in with a valid user ID and PIN before accessing ATM functionalities.

The project aims to demonstrate how ATM systems handle common banking tasks such as **depositing money, withdrawing funds, transferring money, and maintaining transaction history**.

---

## Tools & Technologies Used

* **Java** – Core programming language used for implementing the ATM system
* **Java Collections Framework** – Used to store and manage transaction history
* **VS Code** – Development environment used for coding and testing

---

## Steps Performed

1. **Implemented User Authentication**

   * Created a login system where users enter their user ID and PIN to access ATM functionalities.

2. **Designed a Menu-Driven Interface**

   * Displayed a menu allowing users to choose different ATM operations.

3. **Implemented Transaction History**

   * Stored and displayed previous transactions using an ArrayList.

4. **Developed Withdraw Functionality**

   * Allowed users to withdraw money while checking for sufficient balance.

5. **Implemented Deposit Feature**

   * Enabled users to deposit money into their account and update the balance.

6. **Added Transfer Functionality**

   * Allowed users to transfer money to another account number.

7. **Tested System Operations**

   * Verified correct execution of deposits, withdrawals, transfers, and transaction history updates.

---

## Outcome

The project successfully simulates a **basic ATM system** that allows users to perform common banking operations through a console interface. Users can securely log in, view transaction history, deposit and withdraw money, transfer funds, and exit the system.

This project provided practical experience in **menu-driven program design, object-oriented programming, and handling financial transactions in Java applications**.
